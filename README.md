# scRAMBLE

Semantic Compression for AI Dialogue

## Installation

From source:
```bash
# Clone the repository
git clone https://github.com/yourusername/scramble.git
cd scramble

# Install in development mode
pip install -e .
```

## Usage

Start the interactive CLI:
```bash
ramble
```

Or run as a module:
```bash
python -m scramble
```

## Development

Requirements:
- Python 3.8+
- sentence-transformers
- click
- numpy
